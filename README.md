## Run Locally

Copy `.env.example` to `.env` for usage, and configure your secrets.

```bash
cp .env.example .env
```

Install dependencies & Build

```bash
pnpm install
```

```bash
pnpm build
```

Start the server

```bash
pnpm preview
```
